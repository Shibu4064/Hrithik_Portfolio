# Kanika Chakraborty Portfolio Website

This is a ready-to-host personal portfolio website designed to attract PhD supervisors, academic collaborators, and relevant recruiters.

## Files

- `index.html` — main website
- `css/styles.css` — full responsive styling
- `js/main.js` — interactions, tabs, dark mode, copy buttons, animations
- `assets/kanika-profile.webp` — profile image
- `assets/ielts-scholarship-poster.jpg` — advertisement poster
- `assets/Kanika_Chakraborty_CV.pdf` — downloadable CV
- `assets/favicon.svg` — website icon

## How to host on Netlify Drop

1. Unzip the folder.
2. Go to Netlify Drop.
3. Drag and drop the whole `kanika_portfolio` folder, not only `index.html`.
4. Netlify will give a live website link.
5. Rename the site from Netlify settings if needed.

## What to edit

Open `index.html` and update:

- research interests
- contact information
- publications
- external links such as LinkedIn, Google Scholar, ORCID, ResearchGate
- final PhD topic statement

Sensitive details such as passport number, exact home address, and reference phone numbers are intentionally not displayed.
